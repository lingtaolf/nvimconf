# Personal nvim config for C++ developing  
After plugin installed，what you can do is like below:    
##  File browser  
You can use  `space + e` to open and close file browser  
![image](https://user-images.githubusercontent.com/87509571/158375759-c4bdcf49-8ff6-4c6e-a210-e3f016b50871.png)  
`ctrl+w+w` will help you jump between different window.   
   
## Find file
If you want search some spcified file, you can use `space + f`  
![image](https://user-images.githubusercontent.com/87509571/158376070-affd506d-2fab-46f8-af42-4caad46b3e9b.png)  
  
nvim buffer will show the file you recently opened, the shortcut is `space + b`  
![image](https://user-images.githubusercontent.com/87509571/158376388-8918a928-55ab-4ea6-99ac-41b55cc20598.png)
  
## Code jump  
Nvim support code jumping like `forward` and `back` in IDE.  
Forward is `ctrl+i`, `back` is `ctrl+o`  
  
Go to definition: `gd`  
Go to implementation: `gi`  
  
## Code Compeletion  
![image](https://user-images.githubusercontent.com/87509571/158377225-34a6d8e3-de23-4389-8b1c-af1cbf9ad260.png)
  
Use `tab` to choose the compeletion you want. `shift+tab` to choose upward.
